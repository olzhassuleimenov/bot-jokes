# Bot Jokes
A simple Telegram bot that fetches daily jokes from the web and sends them to users.
This project uses the following libraries:
- `pyTelegramBotAPI`: To interact with the Telegram Bot API.
- `requests`: To make HTTP requests to the website.
- `beautifulsoup4`: To parse the HTML content of the website.
- `python-dotenv`: To manage environment variables (security).
